Robotics Club Summer Projects 2024

Gyrobot : A self-balancing robot

This project aims at building a two-wheeled self-balancing robot that has locomotion capabilities and can maintain its balance just like humans. It can, in fact, achieve perfect equilibrium.

Learning Outcome

  1. Control Theory (PID Control)
  2. Working with Arduino
  3. Design(CAD Design, 3D Printing)
  4. Designing

    5. Space Station : Controls and communication

    This project aims at developing and validating the communication and control system of a Rover over large distances with precision in movements and efficiency in data transfer.

    Learning Outcome

    1. Working with Arduino
    2. Control Theory (PID Control and feedback loops)
    3. Communication using Zigbee
    4. Robotic Operating System (ROS)

    Autonomous Navigation of Mars Rover

    This project aims at designing and simulating the autonomous navigation for the Mars Rover. It emphasises on obstacle detection and classification, obstacle avoidance and path planning to a specific location.

    Learning Outcome

    1. Robotic Operating System (ROS)
    2. Computer Vision
    3. Machine Learning and Deep Learning
    4. Path planning and Localization of the rover

    Robotic Arm

    This project aims at developing a Rover Arm that will be able to pick and place a randomly placed object autonomously. In order to acheive this, it emphasises on learning about the functioning of ROS, Arduino and also controlling motors using microcontrollers. Besides, this project uses inverse kinematics to ensure the arm efficiently converges on the target.

    Learning outcome

    1. Robotic Operating System (ROS)
    2. Inverse Kinematics
    3. PID Control and IMU Calibration
    4. OOPS and Python
